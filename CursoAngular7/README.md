# Actualizar Nodejs
    #### Descargar NODEJS de su sitio web
    ```
    npm install npm@latest -g
    ```    
# Instalar Typescript
    ```
    npm install -g typescript
    ```    
# Instalar Angular
    #### Visitar: cli.angular.io
    ```
    npm install -g @angular-cl
    ng -v
    ```
# Crear app Angular
    ```
    ng new my-dream-app
    cd my-dream-app
    ng serve
    ```
